# Tech Debt Register

Track technical debt explicitly so it gets prioritized and resolved instead of forgotten. Add entries when you take a shortcut; move them to Resolved when fixed.

**Fix effort:** S = a few hours, M = a day or two, L = a week+
**Priority:** P1 = blocks progress or causes incidents, P2 = causes friction, P3 = cleanup / nice-to-have
